# BRM1000

Test race-tracking site generated from `BRM 1000.gpx` with the
`training-journal` Phase 0/V0 pace-test generator.

- Source GPX: private `training-journal/archive/test-rides/BRM 1000.gpx`
- Route distance: ~1022 km
- Generated HTML: `index.html`
- Public site: https://tomverin.github.io/BRM1000/

The HTML is self-contained except for runtime map/CDN assets used by the V0
generator.
